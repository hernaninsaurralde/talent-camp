Nuestra API de prueba con Node, Express y Mongo

Este documento se carga porque: declaramos la carpeta 'static' en el archivo server.js; creamos esa carpeta y dentro de ella colocamos el presente archivo. Su nombre 'debe ser' index.html para que Express lo cargue automáticamente. Si tenemos un controlador de ruta del estilo app.get("/") el archivo estático se cargará en su lugar.
